Physics.update()

核心運動更新 API。負責處理每幀的重力、位移與碰撞檢查。它是維持遊戲世界物理連貫性的樞紐。

def update(self, keys, world_map): # [功能 1] 處理速度與阻力 self.vel[0] *= self.friction self.vel[2] *= self.friction # [功能 2] 應用重力常數 self.vel[1] += GRAVITY # [功能 4] AABB 碰撞檢測 next_pos = self.pos + self.vel if not self.check_collision(next_pos, world_map): self.pos = next_pos

參數說明： keys 為當前鍵盤狀態字典，world_map 為遊戲方塊資料庫。

超越極限的性能

透過「面剔除 (Face Culling)」與「區塊快取」技術，Game X Engine 在 Python 虛擬環境下依然能達成驚人的渲染效率。即便在白色背景的高飽和模式下，也能維持穩定的 60 FPS。

  • 遮擋剔除： 減少 60% 的不必要面渲染
  • 異步載入： 消除地圖邊界載入抖動
  • 輕量數據結構： 壓縮方塊存儲空間
