# 📦 Game X Engine v4.0 - 極簡 Voxel 開發框架

![Game X Engine Logo](logo.png)

> **"Everything You Need. Infinite Possibilities."**
> Game X Engine 是一個專為 Python 開發者設計的 3D Voxel 渲染與物理引擎。致力於將複雜的 3D 遊戲開發「普及化」，讓每一位創作者都能在純白的畫布上建築宇宙。

---

## 🚀 專案願景
Game X 的核心理念是 **「模組化」** 與 **「高效能」**。我們不只是提供一個遊戲，而是提供一個具備 50 項核心 API 的開發平台。無論是想研究柏林雜訊、AABB 碰撞，還是多執行緒區塊加載，Game X 都能提供最直觀的 Python 實作範例。

---

## 🛠️ 50 項核心功能矩陣 (The 50 Pillars)

### 🏃 1. 物理運動系統 (Physics & Motion)
1. **第一人稱控制**：絲滑的滑鼠視角旋轉與 WASD 移動。
2. **動態重力模擬**：可自定義重力常數 $G$。
3. **跳躍物理**：基於衝量的拋物線跳躍。
4. **AABB 碰撞偵測**：精確的方塊與玩家碰撞箱判定。
5. **腳步聲 API**：根據地表材質觸發不同音效接口。
6. **摔落傷害**：動態計算垂直速度產生的衝擊力。
7. **上帝飛行模式**：切換無重力自由穿梭模式。
8. **水中流體阻力**：模擬水中的浮力與運動遲滯。
9. **游泳邏輯**：空間垂直移動與浮沉控制。
10. **移動慣性**：提供 Friction (摩擦力) 參數，讓停步更自然。

### 🌍 2. 世界生成與生態 (World Generation)
11. **無限區塊管理**：動態 Chunk 載入與釋放機制。
12. **柏林雜訊地形**：多層次噪聲疊加，生成自然山脈。
13. **生物群系系統**：沙漠、森林、山地地形分類。
14. **確定性種子**：輸入相同 Seed 即可復現相同世界。
15. **自動化植被**：隨機生成的樹木與裝飾物。
16. **地底洞穴系統**：3D 雜訊挖掘出的中空結構。
17. **晝夜循環**：基於時間的天空顏色漸變。
18. **體積雲渲染**：動態位移的雲朵層。
19. **地層垂直分佈**：草地、泥土、石頭的層次填充。
20. **動態海平面**：自動填充低窪地區的水源方塊。

### ✨ 3. 視覺特效與渲染 (Visuals & Shading)
21. **方向性光影**：根據太陽位置計算方塊面亮度。
22. **大氣距離起霧**：動態混合背景色，隱藏邊界。
23. **半透明渲染**：支援水面與玻璃的 Alpha 混色。
24. **環境光遮蔽 (AO)**：模擬轉角處的陰影加深。
25. **Sine 波水面動畫**：讓水方塊產生週期性律動。
26. **選取框高亮**：精準標註當前指向的方塊邊緣。
27. **粒子破碎系統**：挖掘方塊時產生的物理碎片。
28. **星空天體系統**：隨時間轉動的星圖與月球。
29. **自定義貼圖支援**：API 化紋理映射接口。
30. **動態 FOV**：衝刺時產生的視覺拉伸感。

### ⛏️ 4. 交互與遊戲機制 (Interaction)
31. **Raycasting 挖掘**：高頻射線精準偵測破壞點。
32. **方塊放置**：智慧計算相鄰面座標進行建造。
33. **物品欄管理**：支持多種類型方塊的切換。
34. **方塊硬度屬性**：不同材質具備不同挖掘時間。
35. **快捷列系統**：鍵盤 1-9 鍵即時選取功能。
36. **動態十字準星**：隨環境自動調整顏色的中心點。
37. **HUD 狀態列**：血條與體力槽的渲染 API。
38. **實體掉落物**：挖掘後產生的可拾取模型。
39. **簡易合成系統**：基礎的合成邏輯預留接口。
40. **世界序列化**：JSON 格式的存檔與讀取機制。

### ⚙️ 5. 系統架構與優化 (System & Optimization)
41. **區塊更新優化**：僅在方塊變動時重新計算頂點。
42. **背面剔除 (Back-face Culling)**：忽略玩家看不見的面。
43. **遮擋剔除 (Occlusion)**：被擋住的方塊不進入渲染管線。
44. **異步載入執行緒**：Thread-based 地圖生成，杜絕卡頓。
45. **開發者控制台**：支援 /tp, /time, /give 指令。
46. **自定義啟動畫面**：專業的 Splash Screen 與淡入特效。
47. **一鍵截圖功能**：自動保存當前螢幕為 PNG。
48. **FPS 鎖定器**：防止過度消耗 CPU/GPU 資源。
49. **錯誤診斷日誌**：自動生成的 log 檔案。
50. **Mod API 接口**：支援外部腳本掛鉤 (Hooks)。
